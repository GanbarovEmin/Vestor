Designed for repeat use

Portfolio work without spreadsheet drift

Vestor keeps the daily investment workflow in one native Mac surface: overview, goals, queue, history, dividends, analytics, notifications, settings, and updates.

Portfolio overview

Market value, day movement, gain/loss, cash, dividends, holdings, and data health are visible immediately.

Goal projection

Projects the target portfolio date from current value, growth, dividend yield, and planned contributions.

Planned buys

Keeps upcoming purchases as a first-class queue with dates, tickers, company names, and target amounts.

Quote refresh

Uses Yahoo Finance chart data and shows quote freshness, missing prices, and refresh state.

Local records

Portfolio files, backups, import presets, change journal, and optional provider cookie stay on the Mac.
